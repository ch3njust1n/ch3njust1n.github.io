Problem Definition
Given an image of my face, manipulate it in some simple way and then output the processed image.

Method and Implementation
I used a simple thresholding algorithm that took an image of me (left) and transformed it into a binary image (right). The algorithm iterates through each pixel in the image matrix and sets the pixel to either white RGB(255, 255, 255) or black RGB(0, 0, 0). On each iteration, if the sum of the components exceeds or equals the sum of the average of the component values of all RGB values (256/2*3 = 384), then each component is set to 255. Otherwise, each component value for that pixel is set to 0.

Experiments
I did not do any experimentation other than thresholding the image.
